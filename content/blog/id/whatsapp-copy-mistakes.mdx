Banyak bisnis sudah punya tombol WhatsApp yang bagus, tapi begitu calon klien membuka chat, semuanya berhenti. Pesan pertama yang dikirim sering kali justru membuat orang langsung diam atau bahkan blokir.

Berikut 5 kesalahan copy WhatsApp yang paling sering terjadi — dan cara memperbaikinya.

## 1. Langsung jual tanpa konteks

**Buruk:**
```
Halo, kami dari Klinik Sehat. Ada promo treatment jerawat bulan ini. Mau booking?
```

**Lebih baik:**
```
Halo, saya lihat Anda bertanya soal jerawat di Instagram kami. Apakah ada yang bisa saya bantu jelaskan?
```

**Kenapa gagal:** Orang belum merasa dikenal. Langsung jual terasa seperti spam.

## 2. Terlalu panjang dan bertele-tele

**Buruk:**
```
Selamat siang, terima kasih sudah menghubungi Klinik Sehat. Kami adalah klinik spesialis kulit yang sudah berdiri sejak 2018 dan telah melayani lebih dari 5.000 pasien. Saat ini kami memiliki promo spesial untuk treatment jerawat...
```

**Lebih baik:**
```
Selamat siang! Ada yang bisa saya bantu terkait layanan jerawat kami?
```

**Kenapa gagal:** Calon klien membaca di ponsel. Mereka ingin jawaban cepat, bukan presentasi.

## 3. Menggunakan bahasa yang terlalu formal atau kaku

**Buruk:**
```
Dengan hormat, kami informasikan bahwa jadwal konsultasi dokter kulit kami tersedia pada hari Selasa dan Kamis pukul 10.00–16.00 WIB.
```

**Lebih baik:**
```
Jadwal dokter kulit kami tersedia Selasa & Kamis, jam 10.00–16.00. Mau saya cekkan slot yang masih kosong?
```

**Kenapa gagal:** Terlalu kaku membuat percakapan terasa dingin. Orang lebih suka bahasa yang natural.

## 4. Tidak memberikan pilihan atau langkah selanjutnya

**Buruk:**
```
Baik, terima kasih. Kami tunggu kabar selanjutnya.
```

**Lebih baik:**
```
Baik, saya catat. Mau saya kirimkan jadwal yang tersedia minggu ini, atau ada pertanyaan lain dulu?
```

**Kenapa gagal:** Tanpa langkah selanjutnya yang jelas, percakapan mati di situ.

## 5. Mengabaikan rasa sakit / masalah calon klien

**Buruk:**
```
Kami punya treatment jerawat terbaik di kota ini.
```

**Lebih baik:**
```
Jerawat yang sudah lama tidak kunjung hilang memang bisa bikin frustasi. Treatment kami biasanya mulai terlihat hasilnya setelah 3–4 minggu. Mau saya jelaskan prosesnya?
```

**Kenapa gagal:** Orang tidak peduli seberapa bagus treatment Anda. Mereka peduli apakah Anda mengerti masalah mereka.

## Checklist pesan WhatsApp pertama

- [ ] Menyebutkan konteks (dari mana mereka datang)
- [ ] Singkat (maksimal 2–3 kalimat)
- [ ] Memberi pilihan atau langkah selanjutnya
- [ ] Menggunakan bahasa yang natural
- [ ] Menyentuh masalah atau keinginan calon klien

## Contoh pesan siap pakai

**Klinik / Kesehatan**
```
Halo, saya lihat Anda bertanya soal jerawat. Apakah ada yang bisa saya bantu jelaskan mengenai treatment kami?
```

**Properti**
```
Halo, terima kasih sudah melihat unit kami. Ada yang ingin saya bantu jelaskan soal harga atau denah?
```

**Jasa / Konsultasi**
```
Halo, saya lihat Anda tertarik dengan program mentoring kami. Mau saya kirimkan informasi singkatnya dulu?
```

## Langkah selanjutnya

Jika Anda ingin meningkatkan respons WhatsApp dari iklan atau website, kami bisa membantu:

- Audit 20 pesan WhatsApp terakhir Anda
- Buat template pesan yang sesuai dengan bisnis Anda
- Latih tim untuk menjawab chat dengan lebih cepat dan natural
